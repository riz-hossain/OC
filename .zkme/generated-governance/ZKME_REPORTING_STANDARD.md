# ZKME Reporting Standard

Every meaningful final report must include a ZKME Report.

## Required Fields

- ZKME artifacts consulted
- Files or surfaces identified via ZKME
- Estimated time saved
- ZKME/source mismatches
- Tests run
- Tests not run and why
- Security notes
- Performance notes
- Database notes
- Multilingual/i18n notes

## Report Template

Use `.zkme/generated-governance/TASK_EXECUTION_REPORT.md` for substantial work.

## Citation Rule

For every edited file, cite the ZKME artifacts or source evidence used to decide the edit.
